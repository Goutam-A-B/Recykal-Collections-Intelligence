# Deploy to Google Sheets (not used)

This repository is **Python-only** and does not deploy any Apps Script to Google Sheets.

Use the Python pipeline instead:

```powershell
cd python
pip install -r requirements.txt
python run.py phase5
```

If you need a Google Sheets deliverable for Task B:
1. Upload `deploy/collections_dashboard.xlsx` to Google Drive
2. Open it with Google Sheets
